# NayePankh Foundation Website

---


## Overview

This is a responsive Front End website created for the NayePankh Foundation Front End Development Internship Assignment.

The website is designed to present the foundation's mission, impact, and volunteer opportunities through a clean and user-friendly interface.

---

## Features

- Responsive Navigation Bar
- Hero Section
- Mission Cards
- Impact Statistics Section
- Volunteer Call-to-Action Section
- Volunteer Application Form
- Hover Animations
- Mobile Responsive Design
- Smooth User Experience

---

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries

---

## Purpose

The goal of this project is to demonstrate Front End Development skills by building a modern, responsive NGO website that provides information about the foundation and encourages volunteer participation.
